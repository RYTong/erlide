Nodes tab

In this tab you can choose which nodes you want to trace.

Nodes panel

Adding and deleting patterns

There are three buttons:
  • New node - adds new entry in table of nodes
  • Remove node - removes selected node from the table
  • Add existing nodes - adds all nodes started from erlide

Editing nodes configuration

You can set following parameters:
  • Enabled - if processes from this node should be traced
  • Name - node's name (short name - set using -sname parameter)
  • Cookie - magic cookie's value used when conecting to this node
Column Type can have one of following values:
  • erlide - this node was started from erlide
  • external - this node was started outside erlide
Besides name and cookie values there is net ticktime value which is set for all nodes from preferences window .

Saving and restoring configuration

There are four buttons:
  • Load nodes configuration... - loads previously saved configuration
  • Delete current node configuration... - deletes current configuration
  • Save current nodes configuration - saves current configuration
  • Save current nodes configuration as... - saves current configuration under new name
After loading or saving configuration it's name appears next to Save current nodes configuration as... button.
