Preferences window
Using this window you can set following:
  • Tracing node name - name of node from which other nodes are traced (this value will be used as short name when starting this node). Default value "tracing".
  • Net Ticktime - Net Ticktime (in seconds) for tracing node. Default is 60, possible values: [4, 1000]
  • Max traces in tree viewer - how many traces can be shown in tree viewer. Default is 100, possible values: [1, 150]
