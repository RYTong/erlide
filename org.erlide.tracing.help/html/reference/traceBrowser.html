Trace browser View

Using this view you can browse all tracing results which are currently loaded into Eclipse. After selecting one its content will be displayed in the tree viewer view .
Each element on the list represents directory that contains files with tracing results (in directory there will be one file for each traced node).

Tree viewer
There are 3 buttons available:
  • Load results from disk... - opens dialog window using which you can select directory that contains tracing results.
  • Remove selected - removes selected item from list
  • Remove all - removes all items from list
